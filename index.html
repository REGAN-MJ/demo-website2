Introduction to html

HTML stands for hypertext markup language. It is the standard markup language for creating web pages and web applications. It is used to structure a web page and its content. it uses various tags and attributes to define the different elements on a web page.

Introduction to CSS

CSS stands for cascading style sheet, it is used to style up webpages create layouts and variations in display for different devices and screen sizes. CSS is used to control the layout of multiple web pages all at once.
